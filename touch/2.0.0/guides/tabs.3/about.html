Using TabPanels in Sencha Touch 2

TabPanels are useful to swapping between different pages of an app, with either top or bottom tabs.

Tags:

  • Sencha Touch
  • Sencha Touch 2.0.0
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.TabPanel', {
            fullscreen: true,
            defaults: {
                styleHtmlContent: true
            },
            layout: {
                type: 'card',
                animation: {
                    type: 'fade'
                }
            },
            items: [{
                title: 'Home',
                html: 'Home Screen'
            }, {
                title: 'Contact',
                html: 'Contact Screen'
            }]
        });
    } // launch
}); // application()
