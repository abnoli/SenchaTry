Ext.Button Example

Buttons also come with a range of different default UIs. Here are the included UIs available (if $include-button-uis is set to true):

  • normal - a basic gray button
  • back - a back button
  • forward - a forward button
  • round - a round button
  • action - shaded using the $base-color (dark blue by default)
  • decline - red
  • confirm - green

/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.Container', {
            fullscreen: true,
            padding: 4,
            defaults: {
                xtype: 'button',
                margin: 5
            },
            layout: {
                type: 'vbox',
                align: 'center'
            },
            items: [{
                ui: 'normal',
                text: 'normal'
            }, {
                ui: 'round',
                text: 'round'
            }, {
                ui: 'action',
                text: 'action'
            }, {
                ui: 'decline',
                text: 'decline'
            }, {
                ui: 'confirm',
                text: 'confirm'
            }]
        });
    } // launch
}); // application()
