Ext.ActionSheet Example

As you can see from the following example, you no longer have to specify an xtype when creating buttons within an ActionSheet, because the defaultType is set to button.

Tags: Sencha Touch Sencha Touch 2.0.0 Ext.ActionSheet actionsheet show()
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        var actionSheet = Ext.create('Ext.ActionSheet', {
            items: [{
                text: 'Delete draft',
                ui: 'decline'
            }, {
                text: 'Save draft'
            }, {
                text: 'Cancel',
                ui: 'confirm'
            }]
        });

        Ext.Viewport.add(actionSheet);
        actionSheet.show();
    } // launch
}); // application()
