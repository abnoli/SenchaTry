Ext.Button Example

Buttons can also have a badge on them, by using the badgeText configuration.

Tags: Sencha Touch Sencha Touch 2.0.0 Ext.Button button badgeText
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.Container', {
            fullscreen: true,
            padding: 10,
            items: {
                xtype: 'button',
                text: 'My Button',
                badgeText: '2'
            }
        });
    } // launch
}); // application()
