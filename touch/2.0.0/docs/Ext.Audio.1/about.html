Ext.Audio Example

Here is an example of the Ext.Audio component in a fullscreen container.

Tags: Sencha Touch Sencha Touch 2.0.0 Ext.Audio audio url
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.Container', {
            fullscreen: true,
            layout: {
                type: 'vbox',
                pack: 'center',
                align: 'stretch'
            },
            items: [{
                xtype: 'toolbar',
                docked: 'top',
                title: 'Ext.Audio'
            }, {
                xtype: 'audio',
                url: 'audio/crash.mp3'
            }]
        });
    } // launch
}); // application()
