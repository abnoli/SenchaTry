Determining when a user clicks the Ext.Img component in Sencha Touch 2

The following example shows how you can determine whether an image was clicked using the Ext.Img component in Sencha Touch 2 by using the listeners config object and listening for the tap event.

Tags:

  • Sencha Touch
  • Sencha Touch 2.0.0
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        var logo = Ext.create('Ext.Img', {
            src: 'http://www.sencha.com/img/sencha-large.png',
            mode: 'element',
            listeners: {
                tap: function (img, evt) {
                    Ext.Msg.alert('You clicked the image');
                } // tap
            } // listeners
        }); // create()
        
        Ext.create('Ext.Panel', {
            fullscreen: true,
            items: logo
        }); // create()
    } // launch
}); // application()
