Setting a loading mask on an Ext.List component in Sencha Touch 2

The following example shows how you can apply a load mask to an Ext.List (list) control in Sencha Touch 2 by setting the masked config option to an Ext.LoadMask object.

/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.List', {
            fullscreen: true,
            itemTpl: '{name}',
            masked: {
                xtype: 'loadmask',
                message: 'Please wait while we fetch some more data... ' +
                    '
Still waiting? Seriously, this dialog won\'t ' +
                    'disappear until you call setMasked(false)'
            }, // masked
            store: {
                fields: ['name'],
                data: [{
                    name: 'Red'
                }, {
                    name: 'Orange'
                }, {
                    name: 'Yellow'
                }, {
                    name: 'Green'
                }, {
                    name: 'Blue'
                }] // data
            } // store
        }); // create()
    } // launch
}); // application()
